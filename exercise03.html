Schreiben Sie eine Estimator-Klasse in welcher Sie np.linalg.lstsq verwenden, um die Modell-Koeffizienten zu fitten.

Führen Sie damit eine multiple lineare Regression auf sämtlichen standardisierten Variablen aus.

Wie gross ist das Bestimmtheitsmass?

Verifizieren Sie das Resultat mit scikit-learn (fit_intercept=False).

class MyEstimator:
    def __init__(self):
        self.coef_ = []
    
    def fit_lstsq(self,X,y):
        coefs, residuals, rank, s =np.linalg.lstsq(X,y)
        self.coef_ = coefs
        
    def score(self, X, y):
        predicted = self.predict(X)
        residuals = y - predicted
        return 1- ((residuals - residuals.mean())**2).sum()/((y-y.mean())**2).sum()
    
    def predict(self, X):
        return X.dot(self.coef_)

'S6','female',]

alphas = np.logspace(-2, 7, 200)

    rr = Ridge(fit_intercept=False, alpha=a)
    rr.fit(X_std, y_std)
    if (i == 0):
        coefs = rr.coef_
